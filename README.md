### [Ping Duration Changer](https://www.nexusmods.com/warhammer40kdarktide/mods/636)
<img width="550" height="282" alt="image" src="https://github.com/user-attachments/assets/34a30c7c-2adc-48b2-9cd1-fc83d23ee9d9" />

Standalone mod that gives you sliders to adjust how many seconds a ping lasts before it fades away. The default ping time can sometimes be too short or too long. This is super simple but gives you total control over how long you want to keep track of tagged enemies or items.
